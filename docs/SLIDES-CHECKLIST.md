# Чеклист проверки слайдов

## Админка (/admin/slides)

- [ ] **Reorder:** перетаскивание карточек меняет порядок визуально
- [ ] **Сохранить:** кнопка активна при изменении порядка; после клика порядок сохраняется в БД
- [ ] **Не сохранять:** сбрасывает черновой порядок к исходному
- [ ] **Добавить:** открывает модалку, загрузка файла, сохранение создаёт слайд
- [ ] **Редактировать:** клик по карточке открывает модалку с данными слайда
- [ ] **Удалить:** удаляет слайд и файл из Storage
- [ ] Модалка закрывается по «Отмена», клику вне, ESC

## Главная (/)

- [ ] Слайды отображаются в правильном порядке (sort_order)
- [ ] Автоплей: смена слайда каждые ~5 секунд
- [ ] Переход плавный (~800ms), без дёрганий
- [ ] Стрелки белые, без фона, chevron-форма
- [ ] Нет затемнения поверх изображений
- [ ] Нет текста (заголовков/подзаголовков) на слайдах

## База данных

- [ ] Таблица `hero_slides` без колонок title, subtitle
- [ ] Есть колонки: id, image_url, sort_order, is_active, created_at, updated_at
- [ ] Индекс на sort_order
- [ ] Выполнена миграция `scripts/migrations/hero-slides-final.sql` (если нужно)
